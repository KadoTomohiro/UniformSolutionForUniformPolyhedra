概要

一様多面体とその双対の任意の精度の解法を提示する。解法は、万華鏡構造で与えられたすべての多面体に対して一様であり、各多面体を個別に「調べる」必要はない。

1. 導入

面が正則で頂点が等価な一様多面体は、古代から研究されてきた。最もよく知られているのは、5つの正多面体と13のアルキメデスの立体である。次に，一様な角柱と反角柱の2つの無限のファミリーがある。星型の面や頂点を考慮すると，4つのケプラー・ポアンソ正星形多面体と，1880年代から1930年代に発見された53個の非凸一様多面体の列がありる。この完全なセットは 1953年に Coxeter, Longuet-Higgins and Miller ([CLM], [S]も参照のこと)の論文で初めて示された。

1971年に出版されたMagnus Wenningerの楽しい本 多面体模型 [W1]は、その後何度も復刻されてる。この本には75の一様多面体の段ボール模型の写真と作り方が記載されている。数学的思考を持つ読者がこの本を読むと不思議に思うだろう。「モデルのデータはどのようにして得られるのだろうか? 例えば、1.1600030093は、辺の長さが2の大反屈変形二十・十二面体1のための適切な外接球半径なのだろうか？」

これらのデータが [CLM, 表 7] のデータであることを確認するのは簡単である。いくつかの円周半径は整数と冪乗痕のみで与えられているので正確であるがこれは僅かで、ほとんどは上述のように小数点以下10桁の数字で与えられている。これで十分正確だと思われるかもしれないが、コンピュータのモデリングソフト（[Hy]など）に多面体を組み込む場合は、任意の精度で、あるいはコンピュータが扱える最大の精度で数値を取得したいと思うだろう。さらに，人は多面体の二面体角のような他の幾何学的データの正確な，あるいは最大精度の値に興味があるが，これらについては正則と凸の場合のデータしか利用できず，1′′の精度までである(cf. [CR, Table II]と[J, Table II]を参照)。

この問題はAndrew Humeが扱っている。彼の方法は、そのレポートの以下の短い引用によって最もよく説明されている。[Hm]。

一般に、データは多面体を調べて見つけた方程式の解である (例えば[L, pp.174-176]など)。方程式は記号代数系を用いて3回以上解いた\dots

記号代数系が有用な一様多面体は、いわゆる変形多面体であり、計算には3次方程式や4次方程式を解く必要がある。Humeの解は多面体のデータベースを作成するのに使われ、netlib@research.att.com で公開されている。 symbolic algebra systemsを記号代数系と訳したが何か別のものを指しているかもしれない

私たちのアプローチは全く異なる。個々の多面体を個別に調べるのではなく、以下のようなアプローチを提案する。手持ちの計算機でも理解しやすく使いやすいし、高速な反復アルゴリズムにより任意の精度のデータが得られる可能性があるため、一様多面体とその双対のデータベースを必要としない。さらに、非凸多面体（[Hm]では扱われない）だけでなく、凸多面体に対しても同様に容易に利用できる。ftp@ftp.math.technion.ac.ilで公開されているコンピュータプログラムkaleido (cf. [Ha]) は、 頂点の配置、すなわち、頂点に面として現れる多面体の列挙 (cf. [CR, 2.9.2])、または多面体の万華鏡的構造を記述する、いわゆる Wythoff記号 (cf. [CLM, 3]) のいずれかを与えられた場合に、一様多面体 (およびその双対) のデータを計算するために開発された。また、Kaleidoは、頂点座標と面座標を計算し、各多面体の回転するワイヤーフレーム画像を表示することができる。

2. 基本的なシステム

一様な解法は、一様多面体を同心球上に投影し、球体タイルの各\(n\)辺のタイルを\(2n\)個の合同な球面直角三角形に分解し、三角関数方程式を設定して反復的に解いていくことに基づいている。解法アルゴリズムについては、次項で説明する。ここでは、基本三角形を特定し、基本方程式を設定する手順について述べる。

ここでは、凸の一様多面体を調べているとする。このような多面体は、それぞれの頂点に\(m\)個の面が集まる。\(i\)番目の面は通常シュレフリ記号\(\{n_{i}\}\)で示される正\(n_{i}\)角形であるとする。環状リストlist \((n_{1}, n_{2}, \dots, n_{m})\)は頂点形状である。

多面体を同心球に投影すると、球状のタイリングが得られる。頂点\(C\)を選び、\(i = 1, \dots, m\)の場合の\(A_{i}\)を、\(C\)に集まる1番目のタイルの内心とする。また、\(B_{i}\)を、1番目のタイルと\((i + 1)\)番目のタイル(\((m + 1)\)番目が最初に識別される)を分ける円弧に対し、\(A_{i}\)から下ろした垂直な足との交点とする。実際には、球形多角形\(A_{1}A_{2} \dots A_{m}\)は双対タイル張りの原形タイルである（[GS,§1.2]参照）。また, 平面多角形\(B_{1}B_{2} \dots B_{m}\)は\(C\)のいわゆる頂点図形である (cf. [C, 2.1]).

このようにして球面タイリングを分解し、基本三角形と呼ばれる直角な球面三角形にする。各頂点\(C\)には、\(i = 1, \dots, m\)のために、\(A_{i}B_{i-1}C\)と\(A_{i}B_{i}C\)の対に配置された\(2m\)個の基本三角形が得られる。基本三角形の集合を基本三角形化と呼ぶことにする。 同じ三角形化でも双対タイリングの同様の分解から生じることに注意してください（[GS], 2.7節参照）。

\(A_{i}B_{i}C\)の辺と角度を自明の如く\(a_{i}\), \(b_{i}\), \(c_{i}\), \(\alpha_{i}\), \(\frac{\pi}{2}\), \(\gamma_{i}\)で表す(直角は\(B_{i}\)にある)。\(A_{i}\)の定義と面の規則性から、\(i = 1, \dots, m\)について次のことが容易にわかる。 \[ n_{i}a_{i} = {\pi} \tag{1} \] さらに、\(C\)で出会う基本三角形の角度を考えると、次式が得られる。 \[ \sum \gamma_{i} = \pi \tag{2} \] 最後に、辺\(CB_{i}\)が2つの隣接する基本三角形\(A_{i}B_{i}C\)と\(A_{i+1}B_{i}C\)によって共有されているという事実を利用して、\(a_{i} = a_{i+1}\)を得る。このようにして、すべての\(a_{i}\)は等しい。共通の値\(a\)は、余弦の球面の法則や、円形部分のネピアの第二法則から容易に推論できるように、\(i = 1, \dots, m\) の場合に次式を満たす（[M]参照）。 \[ \cos a = \frac{\cos\alpha_{i}}{\sin\gamma_{i}} \tag{3} \]

このようにして、\(2m + 1\)の未知数\(\alpha_{i}\), \(\gamma_{i}\), \(\cos a\)の\(2m + 1\)連立方程式(\(m + 1\)は線形、\(m\)は三角法)の系が得られる。これを基本系と名付け、次節で解法を説明する。

基本系が解けたら、基本三角形の残りの辺を同じような公式を使って解くことができる。 \[ \cos c_{i} = \frac{\cos \gamma_{i}}{\sin \alpha_{i}} \] \[ \cos b_{i} = \frac{1}{\tan \alpha_{i} \tan \gamma_{i}} \]

基本三角形の解は、一様多面体のすべての幾何学的性質を求める問題を直接反映している。各要素を下記の通り定義する。

\(R\)
外接球半径（中心から頂点までの距離）
\(\rho\)
稜接球半径（中心から稜までの距離）
\(r_{i}\)
内接球半径（中心から面までの距離）
\(l\)
半稜線
\(h_{i}\)
面の内接円半径
\(\theta_{i}\)
面角
\(\phi_{i}\)
二面角（\(i\)面と\(i + 1\)st面との間の角度）
すると下式を検証することは難しくない。
  • \(\frac{\rho}{R} = \cos a\)
  • \(\frac{r_{i}}{\rho} = \cos c_{i}\)
  • \(\frac{l}{\rho} = \tan a\)
  • \(\frac{h_{i}}{r_{i}} = \tan c_{i}\)
  • \(\theta_{i} = \pi − 2\alpha_{i}\)
  • \(\phi_{i} = \pi − c_{i} − c_{i+1}\)

上述した構成は、非凸一様多面体2に対応するように容易に調整することができる。このような多面体は、星型多角形を面として持つこともあれば、頂点で出会う面が頂点を複数回取り囲んでいることもある。前者の場合、\(n_{i}\)は分数（分母は星面の密度、[C]参照）とすることができ、後者の場合、(2)の右辺は\(\pi\)の倍数（乗数は星型の頂点の密度）となる。最後に、非凸多面体は次のように逆行面を持つことがある。頂点図形の外接円周上に任意の向きを選ぶ。\(i\)番目の面は、\(B_{i-1}\)と\(B_{i}\)を結ぶ2つの円弧のうち、短い方の円弧が逆向きである場合、逆行していると言う。(少なくとも1つの面は逆行していないと仮定する！) この場合、逆行している\(\{n\}\)を\(\{n'\}\)で表現し、\(\frac{1}{n} + \frac{1}{n'} = 1\)を満たす\(n' < 2\)とする。これにより、必要に応じて、\(\alpha' = \pi - \alpha\) が鈍角になり、\(\gamma' = -\gamma\)が負になる。これらの調整については次節で再度説明する。

[CLM]p.420において、頂点図形の周辺と初等平面三角法を利用して、我々の基本系と本質的に同じ方程式を導き出していることは注目に値する。彼らはまた、消去法で一つの四次方程式を得ることができ、それを使って変形多面体のリストが完全であることを証明している([CLM], 表4参照)。しかし、我々の視点から見ると、このような導出は、一様多面体とその基本三角形の関係を曖昧にしてしまう（この関係は第4節で役に立つだろう）。さらに、消去法は多面体の種類ごとに新たに実行しなければならず、代数的、三角関数的な記号操作を必要とするため、第3節で紹介する反復解のように一様ではない。

2 球体のタイリングは[GMS]の意味での中空タイルからなり([GS]12.3節参照)、基本三角形化は、三角形による球体の多重タイリング、あるいはリーマン面の三角形化([GS]12.4節参照)とみなすことができる)。

3. 反復的な解法

ここでは、よく知られているニュートンの非線形方程式の解法に基づいて、基本系の反復解法を説明する。一般的な考え方は, 後で独立未知数と称する\(2n + 1\)番目の項で未知数の\(2n\)を解いて, 反復近似によって後者を計算することである。このようなアプローチで問題となるのは独立未知数の選択であり、これについては簡単に説明する。

まず(1)から、\(\alpha_{i}\)が\(i = 1, \dots, m\)の場合について下式のように容易に利用できることに気づく。 \[ \alpha_{i} = \frac{\pi}{n_{i}} \tag{4} \]

さらに、\(i = 1, \dots, m\)の場合について、\(\gamma_{i}\)のうちの1つ、例えば\(\gamma_{1}\)が計算されると、残りのすべての\(\gamma_{i}\)が(3)から利用可能になります。 \[ \cos{a} = \frac{\cos{\alpha_{1}}}{\sin{\gamma_{1}}} \tag{5} \] \[ \gamma_{i} = \arcsin(\frac{\cos{\alpha_{i}}}{\cos{a}}) \tag{6} \]

(6)の逆正弦関数は、\((0, \pi )\)の範囲に一見2つの値を持つことに注意する。ただし、鈍角である可能性がある\(\gamma_{i}\)は最大でも1つしかないことは容易に観察され、一般性を損なうことなく、\(\gamma_{i}\)は、\(i = 2, \dots, m\)の範囲ですべて鋭角であると仮定することができます。 意味がよくわからない。\(\gamma\)(\(C\)の角度)は全て鋭角として扱うと言う意味でいい？

したがって(2)は、\(\delta\)が\(\gamma_{1}\)のみのよく定義された関数である下式に書き換えることができる。 \[ \delta = \pi - \sum{\gamma_{i}} = 0 \tag{7} \]

方程式 \(f(x) = 0\) を解くためのニュートンの方法を思い出してみましょう：ルートへの初期近似を選択し、再帰を用いてより近い近似のシーケンスを定義します。 \[ \hat{x} = x − \frac{f(x)}{f'(x)} \tag{8} \]

最初は基本的な三角形がほぼ平面であると仮定します。これにより、\(i = 1, \dots, m\) の初期近似が得られます。 \[ \gamma_{i} = \frac{\pi}{2} - \alpha_{i} \]

この関数は確かに(5)、(6)(及び\(\cos(a) = 1\))を満たすが、これらの\(\gamma_{i}\)の値は実際よりも小くなり(球面三角形の角度の和が\(\pi\)を超えるため)、(7)の\(\delta\)は消えずに正の数になる。このため、これを余剰関数と命名する。余剰を任意に小さくすることができれば反復処理は収束する。収束を保証するためには経験的に、初期値の最大値として\(\gamma_{1}\)を選択する、すなわち\(\alpha_{1}\)を\(\alpha_{i}\)の最小値として選択すること、あるいは(4)の観点から\(n_{i}\)を\(n_{1}\)の最大値として選択することが必要である可能性が示唆されている。

この選択は、逆正弦関数の二価化についての我々の観察から見て、\(\gamma_{1}\)が鈍角になる多面体の場合には明らかなことである。また、一般的なケースではこの選択が他の選択よりも速い収束を保証することが示唆されている。再帰式を構築したら、なぜそうなるのかを見てみましょう。

(8)を使うには、余剰関数を微分する必要があります。(7)から下式が得られます(プライム(\(\prime \))は\(2, \dots, m\)の範囲の和を表す)。 \[ -\frac{d\delta}{d\gamma_{1}} = 1 + \sum'{\frac{d\gamma_{i}}{d\gamma_{1}}} \tag{9} \] しかしながら、(5)と(6)の観点から、 \[ \sin{\gamma_{i}} = \frac{\sin{\gamma_{1}\cos{\alpha_{i}}}}{\cos{\alpha_{1}}} \] となり、対数微分を用いることで下式を得ます。 \[ \frac{d\gamma_{i}}{d\gamma_{1}} = \frac{\tan{\gamma_i}}{\tan{\gamma_{1}}} \tag{10} \] (7), (9), (10)を組み合わせると次式が得られます(\(1, \dots, m\)の完全な範囲で再び和をとります。\(\delta'\)は(9)の微分です)。 \[ -\frac{\delta}{\delta'} = \delta\frac{\tan{\gamma_{1}}}{\sum{\tan{\gamma_{i}}}} \] 最終的に、\(\gamma_{1}\)に必要な再帰にたどり着きます。 \[ \hat{\gamma_{1}} = \gamma_{1} + \delta\frac{\tan{\gamma_{1}}}{\sum{\tan{\gamma_{i}}}} \]

実際には、頂点で出会う面は必ずしも異なっているわけではないことに気付きますので、類似した面をグループ化し、各頂点で多面体が\(\{n_{i}\}\)である\(m_{i}\)個の面を持つことができるという事実を反映して、\(\sum{m_{i}} = m\)とともに上記の式を調整します。さらに、逆行した面を処理するために、最初の面、つまり最大の面が逆行しないように、頂点の図形を方向づける。したがって、\(n_{i}\)は正の有理数であり、\(m_{i}\)は正の整数である。頂点密度\(d\)は無負の整数である。

要約すると、\(n_{1} = \max n_{i}\)である基本システム \[ n_{i}\alpha_{i} = \pi \] \[ \sum{m_{i}\gamma_{i}} = \pi d \] \[ \frac{\cos{\alpha_i}}{\sin{\gamma_{i}}} = \cos{a} \] は、以下のような反復解アルゴリズムを持っています。

ステップ1.
初期近似値\(\gamma_{i} = \frac{\pi}{2} - \alpha_{i}\)を選択する。このとき、\(\alpha_{i} = \frac{\pi}{n_{i}}\)である。
ステップ2.
余剰\(\delta ＝ \pi d - \sum{m_{i}\gamma_{i}}\)を計算します。計算結果が数値的に十分に小さければ終了します。それ以外の場合はステップ3に進みます。
ステップ3.
下記の再起関数を用い、独立未知数\(\gamma_{1}\)への次の近似値を求める \[ \hat{\gamma_{1}} = \gamma_{1} + \delta\frac{\tan{\gamma_{i}}}{\sum{m_{i}\tan{\gamma_{i}}}} \]
ステップ4.
次式を用い、\(\cos{a}\)および\(\gamma_{i}\)(\(i = 2, \dots, m\))を計算します。 \[ \cos{a} = \frac{\cos{\alpha_{1}}}{\sin{\gamma_{1}}} \] \[ \gamma_{i} = \arcsin(\frac{cos{\alpha_{i}}}{\cos{a}}) \]
ステップ5.
ステップ2に戻る

このアルゴリズムの収束性について観察したい。\(\gamma_{1}\)の増分 は、余剰\(\delta\)が小さくなくても非常に小さくなることがある。これは分母の\(\sum{m_{i}}\tan{\gamma_{i}}\)が大きい際に、分子の\(\tan{\gamma_{1}}\)が同様の振る舞いをしない場合に起こります。このような場合、アルゴリズムの収束が遅かったり、発散したりしますが、我々が独立未知数を選択することで、このようなことが起こらないようにしています。ところで、独立未知数として\(\cos{a}\)を選択すると、同様の問題が発生し、（よく知られているラグランジュ乗数と比較して）一見もっともらしく、おそらくよりエレガントに見えますが、これは避けるべきです。数学的に厳密に収束を証明することはこの論文の範囲外である。有限多数の一様多面体のすべてについて経験的収束が満たされなければならないでしょう([CLM], [S])。

いくつかの例外的なケースでは、直接アプローチが失敗することに注意してください。例外的なケースでは、頂点密度dがゼロであり、頂点に入射する面が複数の対になっているため、初期近似でも基本システムを満たすことになりますが、これは既に述べたように、直角球面三角形では\(\alpha_{i} + \gamma_{i} > \frac{\pi}{2}\)となるため明らかに不可能です。これらの多面体は、4節で紹介するWythoff構成を使って直接構築することができない多面体であることは興味深いことです。第5節では、構造の調整を提示していますが、それはひいては基本システムの調整につながり、これらの例外にも対応できるようにしています。

4. 万華鏡

我々は、二面体万華鏡について議論することから始めます。これは2つのヒンジ付き平面鏡で構成されていますこれらの鏡での反射は等長写像のグループを生成しますが、もし万華鏡の二面体角が\(\pi\)の有理約数である場合、つまり\(\frac{\pi}{q}\)の形をしていれば有限です。 ここで、これらの特殊な角度の意味を追加の視点から考えてみましょう。

万華鏡を垂直面で交差させ、この面で中心が角にある円を選択します。万華鏡は円弧を囲んでおり、この円弧が鏡に繰り返し反射することで、円が有限に覆われる場合があります。\(q = \frac{n}{d}\)の場合に\(d\)回被覆が得られることは簡単にわかります。ここで、\(n\)と\(d\)は互いに素な正の整数であり、\(n > d\)です。この場合、万華鏡は位数\(2n\)の二面体群である\(D_{n}\)を生成し、\(\frac{\pi}{q} = \frac{\pi d}{n}\)弧はグループの基本領域である\(a\frac{\pi}{n}\)弧の\(d\)個のコピーで構成されます。

次に、囲まれた円弧上の一般的な位置に点状の物体を置く。そして，鏡の中で反射を繰り返すと，\(2n\)個の点が得られる。これは自然に\(2n\)角形の頂点と見なすことができます。これは当然、\(2n\)角形の頂点と見なすことができ、隣接する頂点は、端点を共有する円弧に属します。\(2n\)角形を\(C_{1}C_{2} \dots C_{2n}\)とします。ここで、\(C_{1}\)は点オブジェクトです。

2つのケースでは、像の数が少なくなることに注意してください。第１のケースは、\(C_{1} = C_{2}\)または\(C_{1} = C_{2n}\)の場合であり、これは、\(C_{1}\)が元の反射鏡のいずれかの固定点であることを意味するか、幾何学的には、点状物体が鏡のいずれかの上に位置していることを意味する。すると、一致する隣接する頂点の\(N\)組が得られ、これは多角形の辺のうち\(N\)個が縮退し廃棄される可能性があることを示している。このように、点状物体が鏡の一つ上に位置する場合、密度\(d\)を持つ正則な\(n\)角形が得られ、慣習的に\(\{\frac{n}{d}\}\)、すなわち\(\{q\}\)で表される。なお、\(\{q\}\)と\(\{q'\}\)が\(1q + 1q' = 1\)を満たす場合、それらは同じ多角形の2つの対向する向きのコピーであることに注意してください。

もう一つは、\(C_{n+1} = C_{1}\)の場合です。これは、\(n + 1\)番目の円弧が\(1\)番目の円弧と一致しているが、反対方向に向いている場合にのみ可能である。したがって、\(C_{1}\)は万華鏡の二等分面に位置し、この面での反射は万華鏡によって生成された二面体群に属します。この場合、縮退した辺はなく、通常の\(n\)角形が2回転置されています。複合体の密度を\(d\)とすると、\(d\)は偶数でなければならず、各多角形の密度は\(\frac{d}{2}\)である。このように、点オブジェクトが二等分線上にあるとき、二重の\(\{\frac{n}{(d/2)}\}\)、すなわち二重の\(\{2q\}\)が得られます。点オブジェクトが似たような位置にあるが、分母\(d\)が奇数の場合、密度\(d\)を持つ単一の規則的な\(2n\)角形、すなわち単一の\(\{2q\}\)をトレースすることを確認するのは簡単です。

他のすべての場合、多角形は不規則な\(2n\)角形ですが、回転のみ（または任意の数の反射のみ）を使用すると、多角形\(C_{1}C_{3} \dots C_{2n-1}\)は規則的な\(\{q\}\)になります。

ここで、3つ目のミラーを追加し、3つの同時平面ミラーで構成される3面体の万華鏡を得ます。有限の等長写像群を生成する万華鏡の二面体角の条件を見つける問題は、1873年にA.H.シュワルツによって提起され、解決されました。すでに見てきたように、これらの角度は\(\pi\)の有理約数である必要がありますが、それだけでは不十分です。シュワルツは、万華鏡の角を中心とした球を交差させて、次のように問題を定式化した。角\(\frac{\pi}{p}\)、\(\frac{\pi}{q}\)、および\(\frac{\pi}{r}\)（\(p\)、\(q\)、\(r\)はすべて1より大きい有理数）を持つすべての凸球面三角形PQRを列挙し、その辺を何度でも反射させると、PQRは球体を有限に覆うようになるという性質を持つ。このような三角形はシュワルツ三角形と呼ばれ、記号(p q r)で表される。角度の2つが\(\frac{π}{2}\)でない限り（この場合、万華鏡は二面体対称群を生成します）、\(p\)、\(q\)、\(r\)は分子として\(2\)、\(3\)、\(4\)、または\(5\)であり、\(4\)と\(5\)が同時に存在することはできません。分母は分子よりも小さくなければならず、これにより非二面体のシュワルツ三角形の数は有限になります。シュワルツ三角形の完全なリストは、[C]、表III、付録Iにあります。

シュワルツの三角形\(PQR\)の中に点状物体\(C\)を入れ、それを繰り返し反射させてると、点状物体の像を頂点をとする多面体が生成できる。先ほどと同様に、2つの頂点が辺を共有する三角形に属していれば隣接しているとみなされ、複数の頂点が頂点を共有する三角形に属していれば同じ面に付随しているとみなされます。前の説明と同様に、多面体が縮退している可能性があることに注意してください。2つの像がは、シュヴァルツ三角形の同様の一致なしに一致する可能性があります。これは、多面体が、必ずしも区別されていない複数の多面体に分割されていてもよいことを意味する。この現象は[CLM]で調べられていますが、ここでは詳しくは触れません (ただし、第 5 節を参照)。

多面体の万華鏡構造は、それを最初に使用したWythoff (1918)にちなんで命名されました。(四次元では、[CLM]、p.406参照)。

非縮退を仮定すると、ワイソフ記号は、トレースされた多面体の位相不変量、つまり、被覆密度\(D\)とオイラー標数\(\chi\)を取得するのに非常に役立ちます。どちらの値も、シュワルツ三角形による球体の多重タイリングに対して簡単に計算できます。Dは、（p q r）の面積と、その辺での反射によって生成された対称群の次数gの式から求めることができます。 \[ 4\pi D = g(\frac{\pi}{p} + \frac{\pi}{q} + \frac{\pi}{r} - \pi) \] または \[ D = \frac{g}{4}(\frac{\pi}{p} + \frac{\pi}{q} + \frac{\pi}{r} - 1) \] \(g\)の値は、\(n\)倍二面体、四面体、八面体または二十面体において、それぞれ\(4n\)、\(24\)、\(48\)または\(120\)であることに注意されたい。これは、[CLM]（p.412,418,425）で指摘されているように、球面多面体が半球または凹面の面を持っている場合を除いて、\((p\ q\ r)\)から導出できる多面体の密度の正しい値でもあります。 （つまり、半球よりも大きい）。前者の場合、平面は中心に入射し、密度は明確に定義されていません。後者の場合、球面多面体と平面は中心の2つの異なる側で発生し、平面多面体の密度は、凹面の数から上の球面多面体の密度を差し引くことによって計算できます（付録IIの表を参照。 私たちの表記では、\(\gamma\)が鈍角の場合、面は凹面です。セクション3を参照してください。）

オイラー標数は、有名なオイラーの公式\(V - E + F = \chi\)によって特定の閉じた表面に描かれたマップの頂点の数\(V\)、辺の数\(E\)、および面の数\(F\)に関連しています。ここでは、\(F = g\)、\(E = \frac{3g}{2}\)、\(V = V_{p} + V_{q} + V_{r}\)となるようにシュワルツ三角形で三角形分割されたDシートリーマン面を検討しています。\(V_{q}\)は角度\(\frac{\pi}{q}\)の頂点の数です。つまり、\(V_{q} = \frac{g}{2n_{q}}\)です。\(n_{q}\)は\(q\)の分子です。したがって、次式が得られます。 \[ \chi = \frac{g}{2}(\frac{1}{n_{p}} + \frac{1}{n_{q}} + \frac{1}{n_{r}} - 1) \] この値は、\((p\ q\ r)\)から導出可能な非縮退多面体にも継承されます（付録IIの表を参照してください。縮退した場合については、セクション5を参照してください）。\(\chi\)は分子のみに依存することに注意するのは非常に興味深いことです。これは、たとえば、\((2\ 3\ 5)\)から導出可能な凸状のプラトンの二十面体と\((2\ 3\ \frac{5}{2})\)から導出可能な星型の角のあるポアンソの大二十面体は、両方ともオイラー標数\(2\)を持ち、したがって閉じた表面として同相であるということを意味します。 しかしそれらの密度は大きく異なります（前者の場合は\(1\)、後者の場合は\(7\)）。

シュヴァルツ三角形\(PQR\)に点\(C\)を置くと、トレースされる多面体が等角になることは明らかです。つまり、そのすべての頂点は、3つの反射によって生成された等長写像群のアクションの下で同等になります。次に、トレースされた面が正多角形になることを保証する\(C\)の位置について説明し、対応する球面タイリングと基本的な三角形分割およびその三角形のプロトタイプ\(A_{i}B_{i}C\)を調べます。前と同じように、\(A_{1}A_{2}\dots\)は双対タイリングの原型タイル、\(B_{1}B_{2}\dots\)は頂点図形になります。

4.1. CがPQRの頂点Pに存在する場合

\(A_{1}\)と\(A_{2}\)をそれぞれ頂点\(Q\)と\(R\)とし、\(B_{1}\)を\(QR\)上の\(P\)からの垂線の足とします。

結果として得られる基本方程式は、\(p = \frac{n}{d}\) とすると、次式となる。 \[ q\alpha_{1} = r\alpha_{2} = \pi \] \[ n\gamma_{1} + n\gamma_{2} = \pi d \] \(PQR\)をその側面で繰り返し反射することにより、頂点\(A_{1}A_{2}A_{3}\dots\)と\(B_{1}B_{2}B_{3}\dots\)のシーケンスを取得します。これらは、\(2n\)項の後に繰り返されます。

結果として得られる多面体は、ワイソフ記号\(p|q\ r\)で表されます。その頂点構成は\((q. r. q. r \dots q. r)\)（\(2n\)項）であり、その頂点密度は\(d\)です。頂点図形と双対原型タイルは\(2n\)角形です。

4.2. Cが角Rの二等分線と向かい合う辺PQとの交点にある場合

\(A_{1}\)、\(A_{2}\)、\(A_{3}\)を頂点\(Q\)、\(R\)、\(P\)とし、\(B_{1}\)と\(B_{2}\)をそれぞれ\(QR\)と\(RP\)の\(C\)からの垂線の足とします。

結果として得られる基本方程式は次の通りです。 \[ q\alpha_{1} = 2r\alpha_{2} = p\alpha_{3} = \pi \] \[ \gamma_{1} + 2\gamma_{2} + \gamma_{3} = \pi \] \(A_{2}\)、\(B_{1}\)、\(B_{2}\)をQRに反射することで、それぞれ像\(A_{4}\)、\(B_{4}\)、\(B_{3}\)を得ます。

結果として得られる多面体は、ワイソフ記号\(p\ q|r\)で表され、その頂点構成は\((p. 2r. q. 2r)\)です。 頂点図形と双対原型タイルは四角形です

4.3. CがPQRの内心にある場合

\(A_{1}\)、\(A_{2}\)、\(A_{3}\)をそれぞれ頂点\(P\)、\(Q\)、\(R\)とし、\(B_{1}\)、\(B_{2}\)、\(B_{3}\)をそれぞれ\(C\)から\(PQ\)、\(QR\)、\(RP\)への垂線の足とします。

結果となる基本方程式は次の通りです。 \[ 2p\alpha_{1} = 2q\alpha_{2} = 2r\alpha_{3} = \pi \] \[ \gamma_{1} + \gamma_{2} + \gamma_{3} = \pi \]

結果として得られる多面体は、ワイソフ記号\(p\ q\ r|\)で表され、その頂点構成は\((2p. 2q. 2r)\)です。 頂点図形と双対原型タイルは三角形です。

Cが変形多面体をトレースする場合

前述の位置関係の場合では、多面体の対称群は反射によって生成されます。特に、多面体は反射性です。ここで、4番目の位置について説明します。これは通常、キラル多面体、つまり回転のみ3で構成される対称群を持つ多面体をトレースします。これは、（二面角万華鏡で行われたように）偶数回の反射下の像のみを考慮することによって行われます。この場合、双対タイリングを直接調べて、実際の構造なしで\(C\)の存在を推測する方が簡単です。

\(O\)をシュヴァルツ三角形\(PQR\)のフェルマー点、つまり辺が等角をなす角とする点とします（存在すると仮定します 4）。側面で\(O\)を反射し、それぞれ角度が\(\frac{2\pi}{3}\),\(\frac{2\pi}{p}\)、\(\frac{2\pi}{3}\),\(\frac{2\pi}{q}\)、\(\frac{2\pi}{3}\)、\(\frac{2\pi}{r}\)の六角形\(A_{1}A_{2} \dots A_{6}\)を取得します。ここで、奇数のインデックスは\(O\)の像を参照し、偶数のインデックスは頂点\(P\)、\(Q\)、\(R\)を参照します。この六角形を偶数回反射すると、球の同面の垂直方向に規則的なタイリング、つまり一様タイリングの双対が得られます。 後者は、それぞれ\(P\)、\(Q\)、\(R\)を中心とする\(\{p\}\)、\(\{q\}\)、\(\{r\}\)で構成され、いわゆる変形面である3つの\(\{3\}\)と交互になっています。\(C\)をこれらの6つの多面体の共通の頂点とします（変形面\(\{3\}\)は合同であるため、\(A_{1}A_{3}A_{5}\)の外接円に位置し、\(i = 1, 2, \dots, 6\)の\(B_{i}\)を \(C\)から\(A_{i}A_{i+1}\)への垂線の足とする。）

基本方程式は結果として次式になります。 \[ 3\alpha_{1} = p\alpha_{2} = q\alpha_{4} = r\alpha_{6} = \pi \] \[ 3\gamma_{1} + \gamma_{2} + \gamma_{4} + \gamma_{6} = \pi \] ここでは、1つの変形面\(\{3\}\)を3回数えましたが、変形面\(\{3\}\)はすべて一致しているので、3回目の変形面\(\{3\}\)を数えました。結果として得られる変形多面体は、ワイソフ記号\(|p\ q\ r\)で表され、その頂点構成は\((3.\ p.\ 3.\ q.\ 3.\ r)\)です。 頂点図形と双対原型タイルは六角形です。

すべての場合において、\(\alpha = \frac{\pi}{2}\)の出現は、対応するタイルが\(\{2\}\)、つまり二角形であることを意味し、破棄されている可能性があることに注意してください。

3 ただし、\((\frac{5}{2}\ 3\ 3)\)と\((\frac{3}{2}\ \frac{3}{2}\ \frac{5}{2})\)は二等辺三角形であるため、反射性の変形多面体が生成されます。 4 \(O\)の存在は、少なくとも\(PQR\)のすべての角度が\(\frac{2\pi}{3}\)より小さい場合、連続性の引数によって検証されます（これは\((\frac{3}{2}\ \frac{3}{2}\ \frac{5}{2})\)および\((\frac{3}{2}\ \frac{5}{3}\ 2)\)には当てはまりません）。

例外的な多面体

多面体は、その面がコヒーレントに方向付けられる場合、つまり、2つの面に共通の稜が、誘導される方向が反対になるように割り当てられる場合に方向付け可能です。隣接していないシュワルツ三角形に属する頂点が一致している場合でも、それらが別個であると見なされると仮定すると、ワイソフ記号が常に向き付け可能な多面体を生成することを確認するのは簡単です。多面体の頂点の数は、万華鏡の対称群の次数を、頂点を共有する隣接するシュヴァルツ三角形のコピーの数で割ることによって簡単に見つけることができます。 たとえば、\(p\ q|r\)または\(|p\ q\ r\)の場合は\(2\)で、\(\frac{n}{d}|q\ r\)の場合は\(n\)です。

ここで、ワイソフ構成が一致する頂点のペアのみを生成する、2つの特定のケースについて説明します。これらは、識別された場合、方向付けできない、または単側面の多面体の頂点です。これらの場合、実際の頂点の数はもちろん予想される数の半分にすぎません。

この議論は、セクション4で説明した、角度\(\frac{\pi}{q}\)(\(q\)が偶数の分母を持つ有理数)の二面角万華鏡の二等分線上の点オブジェクトが、二重の\(\{2q\}\)を生成するという現象に基づいています。

結論として、存在する唯一の非ワイソフ多面体の解について、簡単で独立した議論をします。

5.1 分母が偶数の\(p\ q\ r|\)多面体

角\(R\)での角度\(\frac{\pi}{r}\)（有理数かつ分母\(r\)が偶数）のシュヴァルツ三角形\(PQR\)を考えてみましょう。点オブジェクト\(C\)が\(PQR\)の内心にあると仮定します。特に、\(R\)の二等分線上にあります。これまで見てきたように、\(C\)の像は\(\{2p\}\)、\(\{2q\}\)、\(\{2r\}\)をトレースし、それぞれ中心が\(P\)、\(Q\)、\(R\)になっています。ただし、\(r\)の分母は偶数であるため、\(RP\)と\(RQ\)のみでの反射によってトレースされる、\(R\)を中心とする面は、二重の\(\{2r\}\)であることがわかります。一度横切ると、\(PQR\)は\(R\)の二等分線に反射され、\(P\)と\(Q\)を中心とする面は、\(P\)と\(Q\)の画像を中心とする2つの同様の面に反射されます。したがって、二重の\(\{2r\}\)に加えて、\(C\)には2つの\(\{p\}\)と2つの\(\{q\}\)があり、輪順\((2r.\ p.\ q.\ 2r.\ q.\ p)\)に配置されています。 二重の\(\{2r\}\)を破棄することにより、単一の単側多面体が得られます。非方向性は方向の反転によるものです。面のコヒーレントな方向を見つけようとし、頂点に入射する4つの面を考慮すると、そのうちの2つは逆行する必要があることに気付きます。つまり、頂点の構成は\((p.\ q.\ p'.\ q')\)です。ただし、破棄された二重の\(\{2r\}\)に隣接する面を考慮すると、頂点に入射する4つの面のいずれも逆行することはありません。

興味深いことに、\(C\)での面の列挙は多面体を適切に定義せず、セクション3の直接的なアルゴリズムアプローチは失敗します。しかし、\({2r}\)を考慮に入れ、4.3節で設定された方程式を解くことで、これらの例外的な多面体の幾何学的性質を計算することができます。

オイラー標数は、セクション4で説明した基本値から取得できます。基本的な三角形分割を考慮し、二重の\({2r}\)を削除することにより、エッジと面のバランスは変化せず、頂点の数は\(V_{r}\)だけ減少します（セクション4を参照）。したがって、この場合は、下式のようになります。 \[ \chi = \frac{g}{2}(\frac{1}{n_{p}} + \frac{1}{n_{q}}− 1） \] これらの多面体は球を均一に覆っていないため、密度は明確に定義されていません。

5.2　半多面体\(p\ p'|r\)

角度\(\frac{\pi}{p}\)と\(\frac{\pi}{q}\)が相補的であるシュヴァルツ三角形\(PQR\)を考え、\(q = p' < 2 < p\)と仮定します。辺\(PR\)を\(PQ\)に反映すると、点\(S\)で\(Q\)を超える\(RQ\)の延長に一致する円弧が得られます。球面三角形\(PSR\)の辺はすべて、\(PQR\)によって生成されたグループの対称弧であることが明らかです。したがって、それ自体がシュヴァルツ三角形となります。たとえば\((\frac{p}{2}\ s\ r)\)です。ここで、\(s\)は\(1\)より大きい有理数です。したがって、\(PSR\)の内心Cは、\(PQR\)の側面\(PQ\)が反対の角度の二等分線と交わる点でもあります。 したがって、\(C\)の像によってトレースされた多面体を考慮すると、多面体\(p\ p'|r\)と\(\frac{p}{2}\ s\ r|\)は同じ頂点を持っていることがわかります。 もちろん、頂点の構成は異なります。前者の場合は\((p.\ 2r.\ p'.\ 2r)\)、後者の場合は（p.\ 2s.\ 2r)\)です。ただし、\(s\)の分母が偶数の場合、\(\frac{p}{2}\ s\ r|\)の二重\({2s}\)は上記のように破棄される可能性があり、頂点構成\((p.\ 2r.\ p'.\ 2r）\)、つまり\(p\ p'|r\)が残ります。\({2r}\)は大円であり、同じ方向を逆行するかどうかを指定できるため、頂点構成の最後の項は\((2r)'\)ではなく\(2r\)と書くことができます。{2r}のこの特性は、CがPとQを超えてRPとRQを拡張することによって得られた「月」の内心であるという観察から得られます。そのような多面体がそれらを半分に分離する赤道面を持っているという特性は、それらをhemi-(半)とラベル付けすることを正当化します。 [lune]はフランス語で月だが、ここで何を指すかよくわからない。二つの大円で作られる図形、もしくは半球のことか？

興味深いことに、もし\(s\)が偶数分母を持つならば、\(CPS\) は \(2s\) が奇数分子を持つシュワルツ三角形\((2\ p\ 2s)\)（または\(2s\)と頂点を共有する半多面体) であることに注意してください。ただし、qが有理数で、2より大きく、分子が偶数である唯一のシュワルツ三角形\((2\ p\ q)\)は\((2\ 3\ 4)\)であることが示されるようです。したがって、向き付け可能な唯一の半多面体は\(3\ \frac{3}{2}|3\)であるということになります（[CLM]、p.417を参照）。

先ほどのケースとは異なり\((p.\ 2r.\ p′.\ (2r)')\)ではなく\((p.\ 2r.\ p′.\ 2r)\)と書けば、多面体\(p p'|r\)の幾何学的性質が頂点の配置から計算可能であることを確認するのは簡単です。実際には、アルゴリズムの1ステップだけで正確な結果が得られます。この対比の背後にある幾何学的な理由は、２種類の非方向性多面体の頂点図形に関連している（[CLM]、p.417とp.419を参照）。一般的な\(p\ q\ r|\)が交差する平行四辺形の頂点図形で、その辺と円の中での内接性が与えられていることから決定されていない。ただし、\(p\ p'|r\)の頂点図形は交差した長方形であり、交差した辺が円の中心を通過するという追加情報によって頂点図形が決まります。

ワイソフ記号は\(p\ p'|r\)の向き付け可能な両面カバーを生成するため、そのオイラー標数は基本的なものの半分です。 \[ \chi = \frac{g}{4}(\frac{2}{n_{p}}+\frac{1}{n_{r}} − 1) \]

5.3 最後の多面体\(|3\ \frac{5}{2}\ \frac{3}{2}\ \frac{5}{3}\)

頂点に6つを超える面を持つ唯一の一様多面体、およびワイソフ記号（またはそのわずかな変更）によって構築できない唯一の一様多面体は、頂点構成\((4.\ \frac{5}{2}.\ 4.\ 3.\ 4.\ \frac{5}{3}.\ 4.\ \frac{3}{2})\)を持ちます。この場合も、頂点に入射する面の列挙では多面体が決定されないため、均一解を直接使用することはできません。ただし、シュワルツ三角形\((3\ \frac{5}{2}\ \frac{5}{3})\)と、その最短辺の垂直二等分線（生成されたグループの対称弧）での反射を考慮すると、次のことが示されます（[CLM]、セクション11を参照）。\(|3\ \frac{5}{2}\ \frac{3}{2}\ \frac{5}{3}\)の三角形と星形五角形の反対方向のペアも、\(|3\ \frac{5}{2}\ \frac{5}{3}\)の鏡像で頂点を共有するペアに属します。実際には、両方の星形五角形は両方の変形多面体に属し、役割が交換されており、各三角形はペアの一方の非変形三角形です。最後の一様多面体は、向き付け可能な半多面体の2番目のインスタンスであると簡単に結論付けることができます。さらに、セクション4.4のように変形多面体を解くと、基本方程式（3）、セクション2を使用して余分な正方形の三角形分割を計算することで解を完成させることができます。オイラー標数の奇妙な値は\(\chi = −56\)で、頂点、エッジ、面の実際の数から計算できます。

付録I: シュワルツ三角形

シュワルツの三角形の列挙を容易にするために、カバーリングが単純であるためには、\((p\ q\ r)\)はメビウス三角形でなければならないこと、つまり、完全二面体対称群、四面体対称群、八面体対称群、二十面体対称群の基本領域である三角形\(D_{\frac{1}{n}} = (2\ 2\ n)\)（ただし、 \(n = 2,\ 3, \dots\)), \(T_{1} = (2\ 3\ 3)\), \(O_{1} = (2\ 3\ 4)\)and\(I_{1} = (2\ 3\ 5)\),のうちの一つでなければならないことに気づきます。球の\(d\)回折りカバーを生成するシュヴァルツ三角形は、メビウス三角形の1つの\(d\)コピーに分解できます。これらのプロパティは、ここで説明するように、シュワルツ三角形を合成するために使用できます。

\[ S_{1} = (p\ x\ r_{1};\ y\ v\ w_{1}) \] および \[ S_{2} = (x'\ q\ r_{2};\ u\ y\ w_{2}) \] を2つのシュワルツ三角形（記号にはそれぞれの角度の反対側も含めます）とします。これらの三角形は、共通の辺\(y\)と相補的な角度\(\frac{\pi}{x}\)および\(\frac{\pi}{x'}\)を持ちます。次に、\(S_{1}\)と\(S_{2}\)を共通のエッジに沿って貼り付け、 \[ \frac{1}{r}=\frac{1}{r_{1}}+\frac{1}{r_{2}} \] \[ w = w_{1} + w_{2} \] を設定した新しい三角形 \[ S_{1} + S_{2} = (p\ q\ r;\ u\ v\ w) \] を生成できます。\(r > 1\)の場合、これはシュヴァルツの三角形です。いつものように、\(2S\)は\(S + S\)の省略形です。

いくつかのケースでは、\(S_{1}\)と\(S_{2}\)を複数の方法で貼り合わせて、いくつかの異なるSchwarz三角形を作成できます。特に、\(S_{1}\)と\(S_{2}\)が同じ直角三角形\((p\ q\ 2;\ u\ v\ w)\)の2つのコピーであり、\(p, q > 2\) and \(p \neq q\)の場合、それらを各脚に沿って貼り付けて、面積は同じで周囲長が異なる2つの二等辺三角形（\((q\ q\ \frac{p}{2};\ w\ w\ 2u)\)と\((p\ p\ \frac{q}{2};\ w\ w\ 2v)\)）を作成できます。

メビウス三角形から始めて、二面体シュワルツ三角形\(D_{\frac{d}{n}} = (2\ \frac{n}{d})\) (ただし、\(d = 1, \dots, n − 1\))の数え切れないほどのファミリーと、表1から3にリストされている四面体、八面体、二十面体の三角形を取得します。三角形は面積を増やすことによって配置されます。たとえば、\(I_{d}\)の面積は\(I_{1}\)の面積の\(d\)倍です。同じ面積の三角形が複数ある場合は、周囲長を大きくして並べ、\(I_{2a}\)と\(I_{2b}\)のように文字で区別します。側面は\(T_{1}\)と\(I_{1}\)の脚\(s\)、\(u\)、\(v\)で表されます5。スペースを考慮して、各シュワルツの三角形を小さな三角形に分割するいくつかの方法の1つだけをリストします。興味深いことに、すべての三角形（最小のものを除く）は、1つだけ例外を除いて、常に2つの三角形に分割することができます。三直角三角形\(D_\frac{1}{2}\)は、当尺性のシュワルツの三角形と考えれば、4つ以上に分割することしかできません。リストされている分割は対称的なものです。正三角形\((\frac{5}{2}\ \frac{5}{2}\ \frac{5}{2})\)は、その半分の3つのコピーである直角三角形\((5\ \frac{5}{2}\ 2)\)で囲まれています。適切な対称サブグループに属する三角形にはフラグが付けられます。

5 \(\cos{2s} = -\frac{1}{3}\), \(\cos{2u} = \frac{\sqrt{5}}{3}\), \(\cos{2v} = \frac{\sqrt{5}}{5}\)であることが示される場合があります。
